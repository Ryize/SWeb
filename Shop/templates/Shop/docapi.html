Документация по Rest API платформы SWeb

GET http://sweb.xyz/api/v1/product/
    Ожидает данные: -
    Возвращаемые данные: Список товаров и их характеристик в формате JSON.
    Характеристики товара:
        name = Название товара
        description = Описание товара
        availability = Наличие товара
        price = Цена
        features = Особенности товара'
        category = Категория товара
        warranty = Гарантия(В месяцах)
        manufacturer_company = Компания производитель
        created_at = Дата создания товара
        seller = Продавец
        seller_email = E-Mail для связи покупателей с вами



GET http://sweb.xyz/api/v1/product/*ID товара*
    Ожидает данные: -
    Возвращаемые данные: Характеристики товара в формате JSON
    Характеристики товара:
        name = Название товара
        description = Описание товара
        availability = Наличие товара
        price = Цена
        features = Особенности товара'
        category = Категория товара
        warranty = Гарантия(В месяцах)
        manufacturer_company = Компания производитель
        created_at = Дата создания товара
        seller = Продавец
        seller_email = E-Mail для связи покупателей с вами



POST http://sweb.xyz/api/v1/productCreate/
    Ожидает данные:
        Authorize = Token *Ваш API Token*
        name = Название товара
        description = Описание товара
        availability = Наличие товара
        price = Цена
        features = Особенности товара'
        category = Категория товара
        warranty = Гарантия(В месяцах)
        manufacturer_company = Компания производитель
        created_at = Дата создания товара
        seller = Продавец
        seller_email = E-Mail для связи покупателей с вами
    Возвращаемые данные: Код 201 в случае успеха, иначе код 400


GET http://sweb.xyz/api/v1/news/
    Ожидает данные: -
    Возвращаемые данные: Список новостей и их характеристик в формате JSON.
    Характеристики новости:
        author_news = Автор новости
        title = Название
        content = Контент
        created_at = Дата публикации
        category = Категория



GET http://sweb.xyz/api/v1/product/*ID товара*
    Ожидает данные: -
    Возвращаемые данные: Характеристики новости в формате JSON
    Характеристики новости:
        author_news = Автор новости
        title = Название
        content = Контент
        created_at = Дата публикации
        category = Категория



POST http://sweb.xyz/auth/token/login
    Ожидает данные:
        login = Имя пользователя
        password = Пароль
    Возвращаемые данные: API Token
